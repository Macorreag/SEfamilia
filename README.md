# SEfamilia

SE que permite la reconstrucción de un árbol genealógico.


## Caracteristicas 

Los tipos de familias que este árbol genealógico puede reconocer son:

- [x] Nuclear
- [x] Monoparental
- [ ] Adoptiva
- [x] Sin hijos
- [ ] Homoparental
- [ ] Ensamblada
- [ ] De Hecho
